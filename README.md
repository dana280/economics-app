## ✨ תכונות

### 📊 עקומת התמורה (PPC)
- גרף אינטראקטיבי של עקומת ייצור
- 6 נקודות מוכנות מראש
- חישובים אוטומטיים:
  - הוצאה כוללת (Total Cost)
  - הוצאה ממוצעת (Average Cost)
  - הוצאה שולית (Marginal Cost)
- הדגמה ויזואלית עם צבעים

### 🔢 מחשבון הוצאות אלטרנטיביות
- התאמה אישית מלאה
- שינוי שמות מוצרים
- סליידרים דינמיים
- תוצאות בזמן אמת
- הסברים מפורטים

### 🎓 קוויז אינטראקטיבי
- 4 שאלות בנושא
- משוב מיידי
- הסברים מפורטים
- מעקב אחר ציונים
- אפשרות לנסות שוב

## 🚀 התקנה מקומית

### דרישות מקדימות
- Python 3.7 ומעלה
- pip (מנהל חבילות)

### צעדי התקנה

1. **שכפל את הריפוזיטורי:**
```bash
git clone https://github.com/dana280/economics-app.git
cd economics-app
```

2. **התקן את החבילות הנדרשות:**
```bash
pip install -r requirements.txt
```

3. **הרץ את האפליקציה:**
```bash
streamlit run streamlit_app.py
```

4. **פתח בדפדפן:**
האפליקציה תיפתח אוטומטית ב: `http://localhost:8501`

## 📚 מושגים שהאפליקציה מלמדת

1. **תופעת המחסור (Scarcity)** - משאבים מוגבלים מול צרכים בלתי מוגבלים
2. **הוצאה אלטרנטיבית (Opportunity Cost)** - הערך של הבחירה הטובה ביותר שוויתרת עליה
3. **עקומת התמורה (PPC)** - גרף של כל השילובים האפשריים של שני מוצרים
4. **הוצאה כוללת (Total Cost)** - סך הויתור
5. **הוצאה ממוצעת (Average Cost)** - עלות ממוצעת ליחידה
6. **הוצאה שולית (Marginal Cost)** - עלות יחידה נוספת
